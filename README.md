# Raedy_to_hunt
A useful tool for setting up a server for bug hunting

# Installation
1. ``` git clone https://github.com/Compact-Blue/Raedy_to_hunt.git && cd Raedy_to_hunt ```
2. ``` chmod +x setup.sh && ./setup.sh ```
